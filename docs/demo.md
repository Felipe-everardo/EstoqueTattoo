# Demonstração

Este projeto ainda não possui deploy público configurado.

Para uma demonstração de portfólio, há três caminhos possíveis:

## Opção 1: GIF no README

Gravar um fluxo curto com:

1. Dashboard
2. Cadastro de material
3. Entrada ou saída de estoque
4. Bancada
5. Histórico de movimentações

Salvar o arquivo como:

```txt
docs/images/demo.gif
```

E referenciar no README:

```md
![Demonstração do sistema](docs/images/demo.gif)
```

## Opção 2: Deploy Separado

- Frontend: Vercel, Netlify ou Azure Static Web Apps
- Backend: Azure App Service, Render ou Railway
- Banco: SQLite para demo simples ou SQL Server/PostgreSQL para uma versão mais realista

## Opção 3: Vídeo Curto

Publicar um vídeo no LinkedIn ou YouTube não listado e incluir o link no README.

Para recrutadores, um GIF curto costuma ser suficiente quando o projeto ainda não tem deploy público.
